# Project Beta

Phase 2 of [[Project Alpha]]. The plan is:

1. Inline `![[embeds]]` into the parent note's embedding so questions
   whose answer lives in the embedded note also surface the parent.
2. Index aliases as standalone search hits — today they sit in
   `note_metadata.aliases_json` but aren't searchable on their own.
3. A model-swap flow that re-indexes into a fresh DB when the user
   wants to try a different embedder.

See [[Daily 2026-05-01]] for the dense-lite idea — could fold that
into Beta as a fourth tier.
